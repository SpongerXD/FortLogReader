# Fortnite Logs Reader

Позволяет прочитать логи из игры Fortnite, в красивом виде.

## Установка:

- Установите Python 3.7-3.9
- Скопируйте нужные логи из папки (AppData\Local\FortniteGame\Saved\Logs)
- Запустите LogReader.py
- Дождитесь окончания парсинга логов и откройте Versions.json.
- Приятного чтения ваших логов!
