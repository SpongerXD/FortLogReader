# Fortnite Logs Reader

Позволяет прочитать логи из игры Fortnite, в красивом виде.


## Установка:

- 1. Установите Python 3.7-3.9
- 2. Скопируйте нужные логи из папки (AppData\Local\FortniteGame\Saved\Logs)
- 3. Запустите LogReader.py
- 4. Дождитесь окончания парсинга логов и откройте Versions.json.
- 5. Приятного чтения ваших логов!
